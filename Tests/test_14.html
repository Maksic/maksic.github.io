Test Scenario
ЛР14
· В документе с traceability matrix создать вкладку «Test Scenario»
· Во вкладке «Test Scenario» составить шаблон тест кейса. Обязательные поля:
1) Идентификатор (ID);
2) Краткое название (Summary, Title);
3) Описание сценария (Description);
4) Участники (Primary Actors);
5) Предусловия (Precondition);
6) Основной сценарий (Basic Flow);
7) Альтернативный сценарий (Alternative Flow);
8) Исключения (Exceptional Flow);
9) Статус (Status);
10) Приоритет (Priority);
· Необходимо написать один сценарий, затрагивающий все компоненты, описанные в спецификации Shop Online Personalization.
· Во вкладке «Traceability Matrix» проставить ID тест сценария напротив ID требований, затронутых в сценарии.
